。客户是服务的请求方，服务器是服务的提供方。服务请求方和服务提供方都要使用网络核心部分所提供的服务 。 服务器软件的特点 一种专门用来提供某种服务的程序，可同时处理多个远地或本地客户的请求。 系统启动后即自动调用并一直不断地运行着，被动地等待并接受来自各地的客户的通信请求。因此，服务器程序不需要知道客户程序的地址 。 般需要强大的硬件和高级的操作系统支持 客户软件的特点 ***二、对等方式（P2P）***：两个主机在通信时并不区分哪一个是访问请求方还是服务提供方，只要两个主机都运行了对等链接软件，他们就可以进行平等的连接通信。本质上仍然是C/S方式，只是对等连接中的每一个主机既充当客户机，又充当服务器。 核心部分 核心部分最关键的部件：路由器 。主机和路由器作用不同： 主机 为用户进行信息处理，并像网络发送分组，或从网络接收分组 路由器
